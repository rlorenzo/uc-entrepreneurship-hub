# AGENTS.md

Agent guidance for this repo lives in **[CLAUDE.md](./CLAUDE.md)**. Read it before working here. It applies to every coding-agent CLI (Claude Code, Codex, Antigravity, Gemini, ...), not just Claude.

Key convention: after every **major** code change, request a read-only second-opinion code review from an installed coding-agent CLI before treating the work as done. The per-tool read-only commands are in CLAUDE.md.

For product and visual-design context, see **PRODUCT.md** (strategy, users, principles) and **DESIGN.md** (the UC visual system).
